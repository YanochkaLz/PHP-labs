div
Тег div використовується для створення блочних елементів у HTML.
span
Тег span використовується для визначення тексту звичайного формату в HTML та CSS.
a
Тег a використовується для створення посилань у HTML.
p
Тег p використовується для визначення абзаців у HTML.
h1
Тег h1 використовується для визначення заголовків першого рівня у HTML.
